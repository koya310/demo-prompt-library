# 業務効率化 プロンプトライブラリ 10本

ChatGPT / Claude で即座に使える、プロフェッショナル向けのプロンプト集です。

## 概要

- **対象**: 営業、HR、SNS運用、カスタマーサポート、経営企画など幅広い職種
- **効果**: 月30-40時間の業務削減が期待できます
- **形式**: すべてのプロンプトをコピーして ChatGPT / Claude に貼り付けるだけで即利用可能

## 使い方

1. **Webページを開く**: https://demo-prompt-library.vercel.app
2. **プロンプトを展開**: 各カードの「プロンプトを見る」をクリック
3. **コピーボタンをクリック**: 「📋 プロンプトをコピー」ボタンでクリップボードにコピー
4. **ChatGPT または Claude に貼り付け**: そのまま貼り付けて実行
5. **条件を入力**: `[例: 顧客名]` など角括弧の部分を自分の情報に置き換え

## カスタマイズ

### 独自プロンプトの追加

1. `index.html` で `.prompt-card` ブロックをコピー
2. ナンバー・カテゴリ・タイトルを変更
3. プロンプト本文を `.prompt-body > pre` 内に記載
4. ブラウザで確認

### スタイル変更

`css/style.css` の `:root` セクションで、カラーパレットを編集:

```css
:root {
  --primary: #2D5BE3;      /* プライマリカラー */
  --secondary: #1A1A2E;    /* セカンダリカラー */
  --bg: #F8FAFF;           /* 背景色 */
  --text: #2C2C2C;         /* テキスト色 */
}
```
